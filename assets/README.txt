Instructions pour l'icône de la bannière

1) Copiez votre fichier d'icône depuis :
   C:\Users\basch\OneDrive\Images

2) Collez-le dans :
   c:\Users\basch\OneDrive\site-html\assets

3) Renommez-le en :
   icon.png

Remarques :
- Le format PNG est recommandé (transparence possible). PNG/JPG/SVG fonctionnent.
- Si vous préférez un autre nom, mettez à jour la balise <img src="..."> dans `ad-banner.html`.
- Ouvrez `ad-banner.html` dans votre navigateur pour vérifier l'affichage local.

Si vous voulez, je peux copier le fichier pour vous si vous me donnez le nom exact du fichier dans le dossier Images (ex: mon-icone.png).
